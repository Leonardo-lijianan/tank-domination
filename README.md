# Tank Domination (坦克大战)

## 【操作说明】

### 玩家1：
W/S/A/D - 上下左右移动

J - 射击

### 玩家2（双人模式）：
↑/↓/←/→ - 上下左右移动

Enter - 射击

## 【障碍物说明】
砖墙 - 可被子弹打穿，阻止坦克移动

铁墙 - 不可被子弹打穿，阻止坦克移动

河流 - 子弹可通过，阻止坦克移动

# 关于


## 【敌人坦克】
浅蓝色 - 普通坦克，100分

粉色 - 射击速度快，200分

红色 - 移动速度快，300分

绿色 - 重型坦克，需4次击毁，400分

深蓝色 - BOSS坦克，需6次击毁，1000分
